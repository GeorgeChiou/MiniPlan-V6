# MiniPlan-V6
MiniPlan V6

Use PLA ( total need about 160g )

Servo : Emax ES08MDII x 18 PCS ( +1 for Head )
Screws : Self-tapping screw M2 x 8mm x 100 PCS , M1 x 3mm x 27 PCS ( from servo )
Bearing : 4x8x3 MR84ZZ x 8 PCS
